# Karaca Sample Showcase - Starter Site

This package contains a ready-to-test static website for:

- Fixed Side Scuttle with Deadlight

## How to test locally

1. Extract the ZIP file.
2. Open `index.html` in a browser.
3. For best 3D model compatibility, run a small local server:
   - Open the folder in VS Code
   - Install/use "Live Server"
   - Right click `index.html` > Open with Live Server

## GitHub Pages upload

Upload these folders/files to your `ekusculu.github.io` repository:

- `index.html`
- `products/`
- `assets/`

After upload, the product page URL should be:

`https://ekusculu.github.io/products/fixed-side-scuttle-with-deadlight.html`

## QR Code URL

Use this URL for the QR code:

`https://ekusculu.github.io/products/fixed-side-scuttle-with-deadlight.html`

## Notes

- Internal project codes are not used in the public-facing page.
- Technical drawing and datasheet buttons can be added later when final public PDFs are selected.
