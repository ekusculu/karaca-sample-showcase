Straight Line Marine Wiper System

A heavy-duty linear marine wiper system designed to provide large, controlled wiping coverage for wheelhouse and marine glazing applications.

Overview

Large controlled coverage

Straight line wipers provide linear wiping motion for broad window areas and can be configured according to the required stroke length and field of vision.

Flexible system configuration

The system can be adapted with single, dual or three-arm arrangements depending on casing length, window size and project-specific requirements.

Technical highlights

Mechanism

  • Linear wiping mechanism
  • Adjustable stroke length
  • Casing length based on stroke
  • Single / dual / three-arm layouts

Motor & control

  • Heavy-duty 24V DC motor
  • Self-parking function
  • Independent control panel

Auxiliary systems

  • Optional spray / washer integration
  • Optional de-icing heater
  • Water and heater cable routing
  • Wash / air / heater control

Application range

  • Wheelhouse windows
  • Large marine glazing
  • Flat or shaped glass surfaces
  • Project-specific installations
